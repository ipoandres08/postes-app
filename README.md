# Postes y Medidores - Monorepo

Este repositorio contiene el código fuente completo de la plataforma de gestión de activos en campo.

## Estructura del Proyecto
- `/backend`: API REST en .NET 8 con Clean Architecture.
- `/mobile`: Aplicación móvil offline-first en Flutter.
- `/infra`: Configuraciones de despliegue, Docker Compose, Nginx y scripts de base de datos.
- `/docs`: Documentación técnica y arquitectura.
